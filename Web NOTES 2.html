INHERITANCE

Inheritance is a key feature in CSS, it is based on the ancestor-descendant relationship . Inheritance is the method by which properties are applied not only to a specified element, but also to its descendants. Inheritance relies on the document tree, which is the hierarchy of HTML elements in a page . Descendant elements may inherit CSS property values from any ancestor element enclosing them. opening tag 'span'Example: h1 { color: orange; } h1(opening tag)Johny is a "nice " boy h1(closing tag). Opening Tag 'h1' Johny is a "nice " boy h1(closing tag). closing tag 'span ''
